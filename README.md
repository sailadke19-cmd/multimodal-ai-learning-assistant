# 🎓 Multimodal AI Learning Assistant

An AI-powered learning platform that helps students study smarter. Upload PDFs, lecture videos, audio recordings, or paste text notes — the app automatically generates smart summaries, study notes, MCQ quizzes, flashcards, and provides a conversational AI tutor with voice support.

## ✨ Key Features
- 📄 PDF, video, audio & text processing
- 🧠 AI-generated summaries, notes & flashcards
- 📝 Auto-generated MCQ quizzes with scoring
- 🤖 Conversational AI Tutor (text + voice)
- 📊 Progress tracking with charts & streaks
- 🌐 Multi-language support (English, Hindi, Marathi)
- 🔐 Secure authentication with JWT & bcrypt

## 🛠️ Tech Stack
**Frontend:** React, Tailwind CSS  
**Backend:** FastAPI, Python  
**Database:** MongoDB  
**AI:** Groq (Llama 3.3), Whisper, gTTS
